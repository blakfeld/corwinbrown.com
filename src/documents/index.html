Systems Administration

Server deployments should be consistent, repeatable, entirely automated, and shouldn't contain any hacks. If an architecture doesn't meet all of those criteria, then as far as I'm concerned, the work isn't done.

DevOps is not only an important mindset, it's the future of enterprise computing. The days where SSHing into a machine and doing some things was a reasonable method are ending quickly. Are you prepared for the future?
